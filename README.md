# Shakthi Portfolio

Live site: https://shakthi-portfolio-nine.vercel.app/

Personal portfolio website showcasing my work, projects, and experience in AI, Data Science, and Software Engineering.  
Built with a modern, performance-focused stack and deployed on Vercel.

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

---

## Features

- Clean, minimal dark UI
- Fully responsive across devices
- Smooth animations and transitions
- Project and experience showcase
- Fast load times and SEO-friendly structure
